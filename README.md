# E-commerce Checkout Process – Manual Test Cases

## Overview

This repository contains manual test cases for the **E-commerce Checkout Process**. The objective is to validate the end-to-end checkout workflow of a registered user, ensuring that each step functions correctly under both positive and negative scenarios.

## Scope

The test cases cover the following modules:

* Product Search and Selection
* Add Product to Cart
* View and Update Cart
* Add Shipping Address
* Payment Methods
* Invalid Payment Scenarios
* Order Placement

## Test Coverage

The Excel workbook includes:

* Functional Test Cases
* Positive Test Scenarios
* Negative Test Scenarios
* Test Case ID
* Description
* Preconditions
* Test Steps
* Test Data
* Expected Results
* Actual Results
* Status
* Comments
